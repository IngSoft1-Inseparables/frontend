# Agatha Christie: Dead on the Cards – Frontend

Proyecto creado con [Vite](https://vitejs.dev/).

---

### Instalación

```bash
nvm use
npm install
```

> **Opcional:**
> Si aparece algún error al instalar dependencias, ejecutar:

```bash
npm install react react-dom
```

---

###  Ejecución

```bash
npm run dev
```

---

### Estructura de carpetas

* **components/**
  Contiene los componentes de presentación de la interfaz de usuario (UI).

* **containers/**
  Incluye los componentes con la lógica de negocio y manejo de datos.

* **services/**
  Contiene los módulos externos que proveen funcionalidades de apoyo (por ejemplo, llamadas a APIs o Websocket).
